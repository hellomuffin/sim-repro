VirtualHome & BEHAVIOR-1K — Egocentric Task Rendering

Two embodied household simulators for the Interactive Online Policy Evaluation benchmark, each showing an agent doing multi-step tasks with an egocentric view and an action scene graph. Every video on this site was rendered headless on an 8×H200 cluster — VirtualHome via Unity software-GL in Docker, BEHAVIOR-1K via OmniGibson + Isaac Sim RTX.
